# Counter-Strike Sound Pack for OpenPeon

Classic Counter-Strike radio commands for your coding sessions.

## Sounds

| Event | Commands |
|-------|----------|
| Session start | "Let's go!", "Lock and load!" |
| Task acknowledge | "Affirmative.", "Moving out." |
| Task complete | "Counter-Terrorists win!" |
| Task error | "Negative.", "Enemy spotted!" |
| Input required | "Need backup!", "Report in." |
| Resource limit | "Hostage down!", "Fall back!" |
| User spam | "Stick together.", "Hold position." |

## Install

```bash
peon-ping --install counterstrike
```
